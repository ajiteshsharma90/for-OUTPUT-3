# Terraform AWS S3 Module

## Features

- Create simple S3 bucket
- Versioning-enabled bucket
- Encrypted S3 bucket (Server Side Encryption (SSE) using KMS CMK or S3-Managed keys)
- S3 bucket with lifecycle rules
